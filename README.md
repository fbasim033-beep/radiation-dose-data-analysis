# Radiation Dose Data Analysis

## Project Overview
This project analyzes radiation dose data for 100 patients using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Features
- Generate synthetic radiation dose dataset
- Perform statistical analysis (mean, std, min, max)
- Visualize dose distribution using histogram

## How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib
2. Run the script:
   python analysis.py
